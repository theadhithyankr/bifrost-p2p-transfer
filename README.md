# Bifrost P2P Transfer

React Native mobile app for encrypted peer-to-peer file transfer using WebRTC, Socket.IO signaling, and Expo.

## Key Features
- React Native mobile app for encrypted peer-to-peer file transfer using WebRTC, Socket.IO signaling, and Expo
- Android or JVM project structure is documented with the relevant Gradle, Kotlin, or Java entry points.

## Setup and Run
```bash
git clone https://github.com/theadhithyankr/bifrost-p2p-transfer.git
cd bifrost-p2p-transfer
npm install
npm run start
npm run android
npm run ios
```
